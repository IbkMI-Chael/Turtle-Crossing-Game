# Turtle Crossing Game

A Frogger-inspired crossing game built with Python's 
Turtle graphics library. Cross the road without getting hit!

## Technologies Used
- Python 3
- Turtle (built-in Python graphics library)

## How to Run It
1. Make sure you have Python 3 installed
2. Download ALL these files into the same folder:
   - main.py
   - player.py
   - car_manager.py
   - scoreboard.py
3. Open your terminal in that folder
4. Run: python main.py

## How to Play
- Press **Up Arrow** to move the turtle forward
- Dodge the incoming cars
- Reach the finish line to advance to the next level
- Cars get faster as levels increase
- Game ends if a car hits the turtle

## Features
- Randomised car spawning and movement
- Level system with increasing difficulty
- Collision detection between player and cars
- Score and level tracking on screen
- Game over screen on collision
